# n8n 评测：真正能用的开源自动化工具

**一句话评价：** n8n 是能自部署的最好自动化工具，花 Zapier 零头的价格获得同等级的能力。

![n8n 官网](/images/n8n-homepage.png)

## n8n 是什么？

n8n 是一个开源工作流自动化平台。可以理解为 Zapier，但你能自己部署（免费）或者用他们的云服务。它能连接 400+ 服务，用可视化编辑器搭建复杂的自动化流程。

核心卖点：数据归你、基础设施你说了算，自部署版没有工作流和执行次数限制。

## 我怎么用的

在一台 $5/月的 VPS 上部署了 n8n，搭了三个自动化：
1. 新博客文章发布→自动分享到 Twitter 和 LinkedIn
2. Stripe 收到付款→添加到表格+发感谢邮件
3. RSS 订阅检查→跟踪的博客发新文章时 Telegram 通知

部署花了 20 分钟（含 Docker 配置）。每个工作流 10-30 分钟，看复杂程度。

可视化编辑器很直观，拖拽节点、连线、配置参数。用过 Zapier 的话上手零障碍。

## 优点

- **自部署真的省钱。** 这三个自动化在 Zapier 上要 $30-50/月。n8n 我花 $5 买个 VPS，还跑着别的东西。
- **无执行次数限制。** Zapier $20 套餐每月限 750 次任务。自部署的 n8n 无限工作流无限执行。
- **AI 集成很扎实。** 原生支持 OpenAI、Anthropic 等大模型。我搭了一个用 GPT-4 总结新文章然后自动写 newsletter 草稿的工作流。
- **需要时能写代码。** Function 和 Code 节点可以直接写 JavaScript，可视化编辑器搞不定的代码搞定。
- **社区活跃。** 5 万+ GitHub Star，活跃论坛，大量模板。常见场景有现成工作流可以导入。

## 缺点

- **自部署需要技术基础。** 得懂 Docker 基础、会配 SSL、知道怎么更新。不难但也不是点点鼠标就行。
- **部分集成比较基础。** 长尾的不太流行的服务，集成深度不如 Zapier 的版本。
- **错误处理要自己搞。** 工作流失败了需要自己设监控和告警，Zapier 这方面开箱即用更好。
- **大工作流编辑器会卡。** 30+ 节点的工作流导航变慢，UI 有些吃力。

## 定价

| 方案 | 价格 | 内容 |
|------|------|------|
| 自部署 | $0（只算服务器费） | 全部无限 |
| Cloud Starter | $24/月 | 5000 次执行，10 个工作流 |
| Cloud Pro | $60/月 | 无限工作流，30000 次执行 |

能跑 Docker 的技术型一人公司：自部署闭眼选。$5/月 VPS 拿到无限自动化能力，第一个月就回本。不想管服务器的话，云版本也比 Zapier 划算，执行次数更多。

## 适合谁

技术型一人公司想要强自动化但不想按次付费，n8n 是最佳选择。自部署 20 分钟搞定，第一个月就赚回来。

需要 AI 驱动自动化的也特别合适——LLM 集成比 Zapier 好，而且不用担心任务次数限制。

非技术用户不想碰服务器的不推荐，Zapier 或 Make 上手更简单。

## 替代方案

- **Zapier：** 上手更简单，非技术用户友好，但大规模使用很贵。$20/月只有 750 次任务。
- **Make（Integromat）：** 可视化编辑器类似 n8n，免费版不错，但还是按执行次数收费。
- **Activepieces：** 另一个开源替代，更新但发展快，比 n8n 更简单。
